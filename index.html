Responsive patterns

This project is a collection of responsive patterns built with HTML and CSS.

Unless otherwise stated, the default oldest browser tested against is Internet Explorer 8.

Layout

  • Grid - percentages, floats, offsets
  • Grid - em's, inline-block
  • Off-canvas secondary content css only menu icon
  • flexbox demonstrating order property

Navigation

  • Hide / reveal with button addEventListener, retina image, pixel-density media query
  • Hide / reveal with background overlay exit addEventListener
  • Display navigation with :toggle selector (no JavaScript) IE9+
  • Off-canvas side menu addEventListener, svg with png fallback
  • Top and side (JavaScript events for side navigation do not change on window resize for IE8) addEventListener, matchMedia, harvey.js, Modernizr.load
  • Select input addEventListener

Images

  • Background stretching over three columns IE9+
  • Images in a text article
  • Intrinsic ratios
  • Pixel-density
  • Font icons
